*Crackly* is a font I made by taking photos on my trip to Thailand and then turning all that text into a font. That procces is explained in more detail on the [the website](https://trulle1234.github.io/Crackly/).

## Setting up locally

First clone the repo and install dependencies:

```bash
git clone https://github.com/Trulle1234/Crackly
cd Crackly
npm install
```

Then start the development server:

```bash
npm run dev
```

Finally go to http://localhost:5173
